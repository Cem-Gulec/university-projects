# CmpE 487 Workshop 2
## P2P Python Chat App
### Requirements
- Python 3
- On the same network with other users

### Usage
```bash
    $ python3 chat.py
```

![Main Menu](main-menu.png)

### Testing
You can create your test scenarios at main function of `test.py`
```bash
    $ python3 test.py ?[TARGET_IP]
```
